# 角色中性参考笔记

> 用途：为题库调参与直觉测试提供更中性的角色事实锚点，尽量避免只凭单一名场面或二创印象来理解角色。

## 使用原则
- 先看角色的身份职责，再看其私人动机。
- 区分“面对局势的公开立场”与“角色内在欲望”。
- 不把戏剧性片段直接等同于稳定人格。
- 题库调参时优先参考“长期决策模式”，而非一次性情绪爆发。

## 角色速记
- 艾伦·耶格尔：核心是自由执念、主动夺回选择权、愿为终结循环承担极端代价。
- 阿明·阿诺德：核心是理解、沟通、策略与“为了改变必须舍弃一部分纯真”的自觉。
- 三笠·阿克曼：核心是关系优先、守护欲、在残酷世界里维持少数重要羁绊。
- 艾尔文·史密斯：核心是战略赌博、群体动员、以自己和他人生命换取真相突破口。
- 利威尔·阿克曼：核心是执行、克制、背负死者意志后的低调责任伦理。
- 莱纳·布朗：核心是身份撕裂、罪责承担、不断在“战士职责”和“真实情感”间受压。
- 吉克·耶格尔：核心是以绝望包装的救世方案，把减少痛苦置于延续未来之前。
- 阿尼·利昂纳德：核心是有限自我、清醒疏离、只愿为极少数私人关系投入。
- 贝特霍尔德·胡佛：核心是顺从宿命、在看清后果后仍麻木执行任务。
- 让·基尔希斯坦：核心是普通人的犹疑、成熟、最终对责任的主动靠近。
- 格里沙·耶格尔：核心是被时代与血脉使命推动的人，长期活在悔恨与托付之中。
- 艾伦·克鲁格：核心是潜伏、忍辱、为了更长远自由而接受脏手策略。
- 肯尼·阿克曼：核心是欲望驱动、犬儒洞察、认为人必须依附某种执念才能活下去。
- 威利·戴巴：核心是政治献祭、叙事操盘、以个人死亡换群体战略空间。
- 弗洛克·福斯特：核心是恐惧催生的极端集体主义，强调强力领袖与岛内生存优先。
- 贾碧·布朗：核心是被灌输的敌我观在现实接触中瓦解，进而重建对“人”的判断。
- 多托·匹西斯：核心是秩序、全局、士气重建与以人类生存为先的务实统帅视角。
- 尤弥尔·弗里茨：核心是被奴役、被爱束缚与等待他人替自己结束长期服从。
- 达里斯·萨克雷：核心是制度权威、裁断权、对旧体制的敌意，以及带有私怨成分的政变参与。
- 萨莎·布劳斯：核心是乡村直觉、对具体生活的热爱、在危险中仍保留对弱者的自然同情。

## 新增角色调参提示
- 多托·匹西斯不等于艾尔文的弱化版：他更偏“守成中的高水平务实”，对人类整体利益的优先级更稳定，私人执念更低。
- 达里斯·萨克雷不等于单纯暴君：他既有军法与制度理性，也有明显的复仇快感与权力审判倾向，适合放在高现实、高代价接受度区间。
- 萨莎·布劳斯不应只被简化成“吃货”：她的角色功能是用具体生活感、人情味和直觉勇气抵消叙事中过强的宏大暴力。
